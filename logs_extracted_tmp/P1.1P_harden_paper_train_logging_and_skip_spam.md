# P1.1P — Kill logging TypeError + stop paper_train skip spam

## Situation
Production is on:
```text
commit=423088e
mode=paper_train
real_orders=False
```

Still failing:
```text
--- Logging error ---
TypeError: %d format: a real number is required, not list
Message: '[PAPER_TRAIN_HEALTH] open=%d closed_1h=%d entries_1h=%d target_1h=%d learning_updates_1h=%d status=%s'
```

Also log spam:
```text
[PAPER_TRAIN_SKIP] reason=cost_edge_too_low ...
[PAPER_TRAIN_SKIP] reason=duplicate_candidate_cooldown ...
```
Dozens/hundreds per second from `DUPLICATE_CANDIDATE`.

## Diagnosis
1. The health log still receives `open_positions` list as first `%d` arg somewhere.
2. The previous hotfix probably added a helper, but not all call paths/log args were replaced.
3. Logging with `%d` is fragile here. Replace with already-formatted f-string or named string after casting.
4. Anti-spam order is wrong: cost-edge skip logs happen before duplicate/rate log suppression, so blocked samples still flood logs.

## Goal
Minimal emergency fix:
- zero logging errors
- no skip-log flood
- keep paper_train opening controlled samples
- no live-real changes
- no P1.2
- no strategy rewrite

## Hard rules
- Do not enable real orders.
- Do not touch live_real order execution.
- Do not change real trading gates.
- Do not add Firebase writes for skips.
- Do not log every skipped duplicate.
- Prefer counters + periodic summary.

## Task 2 — Stop skip spam before cost-edge checks

File:
```text
src/services/paper_training_sampler.py
```

Add module-level throttled skip logger:

```python
_LAST_SKIP_LOG_TS = {}
_SKIP_LOG_TTL_S = 30.0

def _log_train_skip_once(reason: str, symbol: str, side: str, bucket: str, source_reject: str, **extra) -> None:
    now = time.time()
    key = (str(reason), str(symbol), str(side), str(bucket), str(source_reject).split("(")[0])
    last = _LAST_SKIP_LOG_TS.get(key, 0.0)
    if now - last < _SKIP_LOG_TTL_S:
        return
    _LAST_SKIP_LOG_TS[key] = now

    extra_s = " ".join(f"{k}={v}" for k, v in extra.items() if v is not None)
    if extra_s:
        extra_s = " " + extra_s

    log.info(
        "[PAPER_TRAIN_SKIP] reason=%s symbol=%s side=%s bucket=%s source_reject=%s%s",
        reason, symbol, side, bucket, source_reject, extra_s
    )
```

Replace direct `[PAPER_TRAIN_SKIP]` logging with `_log_train_skip_once(...)`.

Critical order in `maybe_open_training_sample()`:
1. Normalize symbol/side/bucket/source.
2. Apply duplicate-candidate cooldown.
3. Apply general dedupe window.
4. Apply entries/min and entries/hour caps.
5. Apply open-position caps.
6. Only then run cost-edge checks.
7. Only if allowed, return entry.

This prevents cost-edge from logging hundreds of times for the same duplicate burst.

For rejected samples:
- increment counters every time
- log only via `_log_train_skip_once`
- do not open position

## Task 3 — Add router pre-throttle

File:
```text
src/services/trade_executor.py
```

In `_maybe_route_to_paper_training(...)`, add a local per-symbol/source/side cooldown before calling sampler.

```python
_PAPER_TRAIN_ROUTER_TS = {}
_PAPER_TRAIN_ROUTER_TTL_S = 10.0

def _paper_train_router_allowed(symbol: str, side: str, source: str) -> bool:
    now = time.time()
    source_key = str(source).split("(")[0]
    key = (str(symbol), str(side), source_key)
    last = _PAPER_TRAIN_ROUTER_TS.get(key, 0.0)
    if now - last < _PAPER_TRAIN_ROUTER_TTL_S:
        return False
    _PAPER_TRAIN_ROUTER_TS[key] = now
    return True
```

At start of `_maybe_route_to_paper_training` after symbol/side extraction:

```python
if not _paper_train_router_allowed(symbol, side, source_reject):
    return False
```

Do not log this fast-path skip. It is only anti-spam.
