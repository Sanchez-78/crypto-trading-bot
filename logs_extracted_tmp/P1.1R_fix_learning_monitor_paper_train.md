## Step 3 — Change paper_trade_executor to call the dedicated API
In `paper_trade_executor.py`, update `_safe_learning_update_for_paper_trade()`:

```python
def _safe_learning_update_for_paper_trade(raw_trade: dict) -> bool:
    trade = _canonical_closed_paper_trade(raw_trade)
    try:
        from src.services.learning_monitor import update_from_paper_trade
        ok = update_from_paper_trade(trade)
        if ok:
            log.info(
                "[LEARNING_UPDATE] source=paper_closed_trade symbol=%s regime=%s bucket=%s outcome=%s net_pnl_pct=%.4f ok=True",
                trade["symbol"], trade["regime"], trade["training_bucket"], trade["outcome"], trade["net_pnl_pct"]
            )
        else:
            log.warning(
                "[LEARNING_UPDATE_ERROR] err=update_from_paper_trade_false symbol=%s bucket=%s",
                trade["symbol"], trade["training_bucket"]
            )
        return bool(ok)
    except Exception as e:
        log.exception(
            "[LEARNING_UPDATE_ERROR] err=%s symbol=%s bucket=%s fn=update_from_paper_trade",
            e, trade.get("symbol"), trade.get("training_bucket")
        )
        return False
```

Remove/disable direct call to old `lm_update()` from paper training path.
