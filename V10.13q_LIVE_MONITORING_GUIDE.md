## DATA COLLECTION FOR V10.14 TUNING

**Create a log analysis script** to run after 48 hours:

```bash
#!/bin/bash
# analyze_v10.13q.sh

LOG_FILE="/path/to/bot/logs/trading.log"

echo "=== V10.13q LIVE ANALYSIS ===" > analysis.txt
echo "" >> analysis.txt

echo "TIER PENALTIES APPLIED:" >> analysis.txt
grep "V10.13q_PAIR" $LOG_FILE | sed 's/.*TIER/TIER/' | sort | uniq -c >> analysis.txt

echo "" >> analysis.txt
echo "KILL DISTRIBUTION (last 100 cycles):" >> analysis.txt
grep "V10.13q CYCLE" $LOG_FILE | tail -100 | grep "kills=" | sed 's/.*kills=//' | sort | uniq -c >> analysis.txt

echo "" >> analysis.txt
echo "RESCUE EVENTS (emergency override activations):" >> analysis.txt
grep "EMERGENCY_OVERRIDE\|TIER4_OVERRIDE" $LOG_FILE | wc -l >> analysis.txt

echo "" >> analysis.txt
echo "EXECUTION RATE (trades per cycle):" >> analysis.txt
grep "exe=" $LOG_FILE | tail -50 | sed 's/.*exe=//' | awk '{sum+=$1; count++} END {print "avg=" sum/count " total=" sum}' >> analysis.txt

echo "" >> analysis.txt
echo "BLOCKED_RATIO (estimated):" >> analysis.txt
TOTAL_KILLS=$(grep "kills=" $LOG_FILE | tail -20 | sed 's/.*kills=//' | tr ',' '\n' | awk -F= '{sum+=$2} END {print sum}')
TOTAL_GEN=$(grep "V10.13q CYCLE" $LOG_FILE | tail -20 | sed 's/.*gen=//' | sed 's/ .*//' | awk '{sum+=$1} END {print sum}')
echo "scale=2; $TOTAL_KILLS / $TOTAL_GEN" | bc >> analysis.txt

echo "Analysis saved to analysis.txt"
```
